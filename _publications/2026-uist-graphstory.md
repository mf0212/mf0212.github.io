**Authors:** **Minh-Loi Nguyen\***, Xuan-Vu Le\*, Khanh-Duy Le, Minh-Triet Tran, Trung-Nghia Le.

**Venue:** ACM Symposium on User Interface Software and Technology (UIST 2026). **Under Review.**

GraphStory turns collaborative story writing into editing a shared event graph: contributors add, branch, and merge narrative events instead of overwriting each other's prose. The system surfaces conflicts at the event level, lets writers reason about plot consistency before generation, and makes large language model contributions accountable to the agreed structure.
