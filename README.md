# mf0212.github.io

Personal academic website of **Nguyen Minh Loi** — AI Researcher at Trivita AI, Honor Program at VNUHCM-University of Science, prospective PhD applicant for Fall 2026.

Live site: <https://mf0212.github.io>

## Stack

Built with [Jekyll](https://jekyllrb.com/) and deployed via GitHub Pages.

## Local development

```bash
bundle install
bundle exec jekyll serve
```

The site will be available at <http://localhost:4000>.

## Layout

- `_pages/` — top-level pages (About, CV, Publications, Talks)
- `_publications/` — one Markdown file per paper
- `_talks/` — talks, posters, invited lectures, competitions
- `_data/navigation.yml` — site navigation
- `_includes/`, `_layouts/`, `_sass/` — theme internals
- `images/` — site assets
